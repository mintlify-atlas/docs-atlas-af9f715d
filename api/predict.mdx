## Overview

The prediction module loads a trained model and preprocessor, processes test data, generates predictions, and creates a submission file in the required format.

## Function

### predict()

Loads the trained model and generates predictions for test data.

```python
def predict() -> None
```

The function performs the complete inference pipeline including model loading, data preprocessing, prediction, and submission file creation.

#### Prediction Pipeline

1. **Model Loading**
   - Loads `best_model.joblib` - the trained classifier
   - Loads `preprocessor.joblib` - the fitted preprocessing pipeline

2. **Data Loading**
   - Reads test data CSV
   - Extracts `id` column for submission file

3. **Data Preprocessing**
   - Applies `clean_data()` function
   - Removes `gender` column (not present in test data)
   - Transforms data using fitted preprocessor

4. **Prediction**
   - Generates predictions using the trained model
   - Maps predictions back to original labels: `0 → 1` (Male), `1 → 2` (Female)

5. **Submission Creation**
   - Creates DataFrame with `id` and `gender` columns
   - Saves to `submission.csv`

## Implementation

<Expandable title="Complete predict() function source code">

```python
def predict():
    # Load model and preprocessor
    model = joblib.load(os.path.join(root_dir, "best_model.joblib"))
    preprocessor = joblib.load(os.path.join(root_dir, "preprocessor.joblib"))
    
    # Load test data
    test_df = pd.read_csv(test_data_path)
    ids = test_df['id']
    
    # Process test data (note: gender is missing or zero/null in test set)
    X_test = clean_data(test_df)
    X_test = X_test.drop('gender', axis=1, errors='ignore')
    
    # Transform
    X_test_processed = preprocessor.transform(X_test)
    
    # Predict
    preds = model.predict(X_test_processed)
    # Map back to original gender labels (0 -> 1, 1 -> 2)
    final_preds = [1 if p == 0 else 2 for p in preds]
    
    # Create submission
    submission = pd.DataFrame({
        'id': ids,
        'gender': final_preds
    })
    
    output_path = os.path.join(root_dir, "submission.csv")
    submission.to_csv(output_path, index=False)
    print(f"Predictions saved to {output_path}")
```

## Test Data Format

The test data CSV should contain the same columns as training data (except `gender`):

**Required Columns:**
- `id` - Unique identifier for each sample
- `height` - Height in cm
- `weight` - Weight in kg
- `sleepiness` - Sleepiness score
- `iq` - IQ score
- `fb_friends` - Number of Facebook friends
- `yt` - YouTube-related metric
- `star_sign` - Astrological star sign
- `phone_os` - Phone operating system
- `self_intro` - Self-introduction text

**Notes:**
- Missing values are handled automatically by the preprocessing pipeline
- The `gender` column may be absent or contain null/zero values in test data

## Submission CSV Format

The generated `submission.csv` follows this format:

```csv
id,gender
1,1
2,2
3,1
4,2
5,1
...
```

<ResponseField name="id" type="int">
  Unique identifier from the test dataset
</ResponseField>

<ResponseField name="gender" type="int">
  Predicted gender label:
  - `1` = Male
  - `2` = Female
</ResponseField>

## Label Mapping Reference

The model uses binary encoding internally but the submission requires original labels:

| Original Label | Description | Model Encoding | Submission Output |
|---------------|-------------|----------------|-------------------|
| 1 | Male | 0 | 1 |
| 2 | Female | 1 | 2 |

**Conversion Logic:**
- Training: `gender.map({1: 0, 2: 1})` - converts original to model format
- Prediction: `[1 if p == 0 else 2 for p in preds]` - converts model output to submission format

## Error Handling

```python
import os
import joblib

def safe_predict(test_csv_path, model_path, preprocessor_path, output_path):
    """Prediction with error handling"""
    
    # Check if required files exist
    if not os.path.exists(model_path):
        raise FileNotFoundError(f"Model not found: {model_path}")
    
    if not os.path.exists(preprocessor_path):
        raise FileNotFoundError(f"Preprocessor not found: {preprocessor_path}")
    
    if not os.path.exists(test_csv_path):
        raise FileNotFoundError(f"Test data not found: {test_csv_path}")
    
    try:
        # Load and predict
        model = joblib.load(model_path)
        preprocessor = joblib.load(preprocessor_path)
        
        test_df = pd.read_csv(test_csv_path)
        ids = test_df['id']
        
        X_test = clean_data(test_df)
        X_test = X_test.drop('gender', axis=1, errors='ignore')
        X_test_processed = preprocessor.transform(X_test)
        
        preds = model.predict(X_test_processed)
        final_preds = [1 if p == 0 else 2 for p in preds]
        
        submission = pd.DataFrame({'id': ids, 'gender': final_preds})
        submission.to_csv(output_path, index=False)
        
        print(f"Success! Predictions saved to {output_path}")
        return submission
        
    except Exception as e:
        print(f"Error during prediction: {str(e)}")
        raise
```

## Notes

- **Preprocessor Required**: Always use the same preprocessor that was fitted during training
- **Data Cleaning**: Test data must go through `clean_data()` before preprocessing
- **Label Mapping**: Remember to convert model predictions (0/1) back to submission format (1/2)
- **ID Column**: Ensure the `id` column is preserved for the submission file
- **No Fitting**: Never call `fit()` on the preprocessor with test data - only use `transform()`

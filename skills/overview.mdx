## What are Skills?

The Boy or Girl Predictor project uses **Google Antigravity's agent-based workflow system**. Skills are specialized AI agents that focus on specific aspects of the data science and machine learning pipeline. Each skill has domain expertise and follows best practices for its particular area.

Instead of writing all code yourself, you orchestrate these specialized agents by invoking them with the `@` prefix (e.g., `@data-scientist`, `@ml-model-trainer`).

## Core Concept

The agent-based workflow breaks down complex ML projects into specialized roles:

- **Separation of Concerns**: Each skill focuses on one aspect of the ML pipeline
- **Best Practices Embedded**: Skills follow industry standards and engineering principles
- **Orchestration**: You combine skills to build complete workflows
- **Consistency**: Skills reference the Implementation Playbook for production-grade code

## How to Invoke Skills

To use a skill, simply mention it with the `@` prefix in your prompt:

```
@data-scientist create a baseline model for gender prediction
```

```
@ds-data-cleaner audit the training dataset for missing values
```

```
@ml-model-trainer tune XGBoost hyperparameters with 50 trials
```

You can also chain skills together:

```
First use @ds-data-cleaner to audit the data, then @data-scientist to build a baseline
```

## Risk Levels

Each skill has an associated risk level:

- **Medium Risk**: Skills that modify data or create experimental code (`@data-scientist`, `@ds-data-cleaner`)
- **High Risk**: Skills that affect production systems or resource-intensive operations (`@ml-engineer`, `@ml-model-trainer`)

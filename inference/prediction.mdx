## Overview

Once your model is trained, you can use it to make predictions on new test data. The prediction process involves loading the saved model and preprocessor, preparing the test data, and generating gender predictions.

## Gender Label Mapping

The model uses an internal representation that differs from the competition labels:

- **Model Output**: `0` (Male) or `1` (Female)
- **Competition Labels**: `1` (Male) or `2` (Female)

<Warning>
Always remember to map the model predictions back to competition format: `0 -> 1` (Male) and `1 -> 2` (Female)
</Warning>

## Prediction Process

<Steps>
  <Step title="Load the trained model and preprocessor">
    Use `joblib` to load both the model and preprocessor that were saved during training.

    ```python
    import joblib
    import os

    # Load the saved artifacts
    model = joblib.load(os.path.join(root_dir, "best_model.joblib"))
    preprocessor = joblib.load(os.path.join(root_dir, "preprocessor.joblib"))
    ```

    The preprocessor contains the fitted transformations (scaling, encoding) from the training data.
  </Step>

  <Step title="Load and prepare test data">
    Load the test CSV and extract the IDs for submission. Apply the same cleaning pipeline used during training.

    ```python
    import pandas as pd
    from data_pipeline import clean_data

    # Load test data
    test_df = pd.read_csv(test_data_path)
    ids = test_df['id']
    
    # Clean the test data
    X_test = clean_data(test_df)
    X_test = X_test.drop('gender', axis=1, errors='ignore')
    ```

    Note that the test set doesn't have gender labels, so we remove that column if it exists.
  </Step>

  <Step title="Transform test features">
    Apply the preprocessor to transform the test data using the same scaling and encoding from training.

    ```python
    # Transform test data using fitted preprocessor
    X_test_processed = preprocessor.transform(X_test)
    ```

    <Warning>
    Always use `transform()` on test data, never `fit_transform()`. The preprocessor should only be fitted on training data.
    </Warning>
  </Step>

  <Step title="Generate predictions">
    Use the trained model to predict gender labels and map them to competition format.

    ```python
    # Generate predictions
    preds = model.predict(X_test_processed)
    
    # Map model output to competition labels
    # 0 -> 1 (Male), 1 -> 2 (Female)
    final_preds = [1 if p == 0 else 2 for p in preds]
    ```
  </Step>
</Steps>

## Complete Prediction Script

Here's the full prediction code from `predict.py`:

```python
import pandas as pd
import numpy as np
import os
import joblib
from data_pipeline import clean_data

# Root directory
root_dir = "/home/gk/github/gk-work/DSML/IM5033-boy-or-girl-2026"
test_data_path = os.path.join(root_dir, "data/boy-or-girl-2025-test-no-ans-missingValue.csv")

def predict():
    # Load model and preprocessor
    model = joblib.load(os.path.join(root_dir, "best_model.joblib"))
    preprocessor = joblib.load(os.path.join(root_dir, "preprocessor.joblib"))
    
    # Load test data
    test_df = pd.read_csv(test_data_path)
    ids = test_df['id']
    
    # Process test data (note: gender is missing or zero/null in test set)
    X_test = clean_data(test_df)
    X_test = X_test.drop('gender', axis=1, errors='ignore')
    
    # Transform
    X_test_processed = preprocessor.transform(X_test)
    
    # Predict
    preds = model.predict(X_test_processed)
    # Map back to original gender labels (0 -> 1, 1 -> 2)
    final_preds = [1 if p == 0 else 2 for p in preds]
    
    # Create submission
    submission = pd.DataFrame({
        'id': ids,
        'gender': final_preds
    })
    
    output_path = os.path.join(root_dir, "submission.csv")
    submission.to_csv(output_path, index=False)
    print(f"Predictions saved to {output_path}")

if __name__ == "__main__":
    predict()
```

## Running Predictions

Execute the prediction script from your terminal:

```bash
python predict.py
```

You should see output confirming the predictions were saved:

```
Predictions saved to /path/to/submission.csv
```

## Understanding the Output

The prediction process outputs:
- Raw model predictions (0 or 1)
- Mapped competition labels (1 or 2)
- A DataFrame with `id` and `gender` columns ready for Kaggle submission

</Tip>
